                         Quick Reference



    Navigation

        Click any pane             Focus pane (others dim)
        Drag pane border           Resize panes



    Pane Actions

        │ Split                    Add column (side-by-side)
        ─ Stack                    Stack vertically (1 per column)
        × Close                    Close current pane
        ⎋ Exit                     Stop tool, return to shell



    Tools

        opencode                   Launch OpenCode AI
        claude                     Launch Claude Code



    Tips

        Scroll                     Mousewheel or trackpad
        Select text                Click and drag
        Stop command               Ctrl + C
        Close shell                Type 'exit'




                                Press q to close
